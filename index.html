curious coders, here's how it works

  • The site structure is based off the fantastic HTML5 boilerplate.
  • Fonts are provided by Google webfonts.
  • The filter functionality is from David Desandro's impeccable Isotope jQuery plugin.
  • The syntax highlighting is done by the neatly minimal prettify.
  • The copy-to-clipboard feature is done with z clip, which uses an invisible flash (.swf) file.
  • User preferences saved to browser via HTML5 localstorage
  • The snips are loaded dynamically through AJAX, from an XML file.
  • Enhancements include CSS3 transition animations + box shadow.
